Football League Simulator Project

Language: python

Library/environment: processing

I have always been interested in sports and statistics, so this project was an attempt to combine the two to create fun game. I started this project during the covid lockdown after graduating high school and spent most of the summer working on it. I worked on and off on this project for the next year or two before finally deciding the code had become too much of a mess to maintain. I'm happy with the product, but this is also taught me a lesson about making sure my code is maintainable.
