Football League Simulator Project

Language: Python

Environment/Libraries: processing

Things I Learned: OOP, UI design, importance of organizing code

GitHub: https://github.com/cost12/Football-Game

I have always been interested in sports and statistics, so this project was an attempt to combine the two to create fun game. I started this project during the covid lockdown after graduating high school and spent most of the summer working on it. I worked on and off on this project for the next year or two before finally deciding the code had become too much of a mess to maintain. I'm happy with the product, but this is also taught me a lesson about making sure my code is maintainable. Unfortunately, this project is difficult to share, so instead I have some screenshots highlighting the game's features.
